Firewalls

  • Protecting  your computer from the big bad internet…
  • Hardware or software based
  • Checkpoint for communication on a network
  • Blocks unauthorized access


For home is -> All-in-one : firewall, router, switch, and wireless access point

Technologies

  • Packet Filtering (control network access by monitoring outgoing and incoming packets)
  • Circuit level gateways (monitor TCP handshaking between packets to determine whether a requested session is legitimate)
  • Application level gateways (client program establishes a connection to a destination service)
  • Statefull Inspection ((SPI), also referred to as dynamic packet filtering)

DMZ
purpose of a DMZ is to add an additional layer of security to an organization's local area network (LAN); an external network node only has direct access to equipment in the DMZ


Network Intrusion

  • Unauthorized access to the network
  • Virus, Worms and Trojans replicating on the network
  • Specially crafted network packages sent to exploit a vulnerability
  • Attacks to render services unresponsive
